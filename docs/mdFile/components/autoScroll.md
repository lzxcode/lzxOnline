# 基础用法

通过 height 属性设置滚动高度，若不设置则默认 200px

components/autoScroll/basic
:::

# API

# 属性

| 属性名 |    说明    |   类型 | 默认值 |
| ------ | :--------: | -----: | ------ |
| height | 滚动的高度 | string | 200    |
| mode   |  滚动模式  | string | auto   |
